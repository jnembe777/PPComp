⚠ Correction théorique fondamentale — Dimension B : Les formules L1 = r·log m, L2 = logN + N·log(rm), L3 = r + N·log m, L4 = logN + logC(r,N) + N·log m ne sont valables que lorsque le codage de la couleur est uniforme (mode Bb). Le terme N·log₂m est le coût couleur uniforme. Ce terme est remplacé par C_color(B) selon le mode :
Ba = séquentiel : 1 terme en tête de bloc uniquement (+ transitions explicites) · Bc = Huffman : N·H(P̂) + D_huf ≤ N·log m · Bd = Elias universel : N·L*(m) si m non fixé. Le codage couleur devient la dimension B du cartouche ABCDEFGH.

Agent Codage Couleur — Dimension B ★ (nouvelle)
Ba Séquentiel · Bb Uniforme (L1–L4 actuels) · Bc Huffman N·H(P̂) · Bd Elias universel
Inactif
Phase 2 · Dim. B — Correction fondamentale
Ba — Séquentiel : la couleur n'est indiquée qu'une seule fois en tête de bloc. Chaque saut hérite de la couleur courante. Seules les transitions explicites (τ_k, nouvelle_couleur) sont codées. Coût : log₂m + N_trans·log₂m bits.
Bb — Uniforme : chaque saut porte sa marque codée uniformément sur m couleurs. N·log₂m bits. C'est le terme dans L1–L4 actuels.
Bc — Huffman : codes de longueur variable sur P̂(m_k). Coût = N·H(P̂) + D_huf. Avantageux si N > N* = D_huf / (log₂m − H(P̂)).
Bd — Universel (Elias) : si m n'est pas connu au décodeur. Coût ≈ N·(log₂m + 2·log₂log₂m), proche de l'entropie si m grand.

Dimension B — 4 modes de codage couleur
Mode BTerme couleur C_color(B)Quand optimalLien L1–L4
Ba — Séquentiel log₂m + N_trans · log₂m Processus couleur lentement variable (N_trans ≪ N) N_trans bits de transitions vs N·log₂m uniforme
Bb — Uniforme N · log₂m ← formules L1–L4 Distribution P̂ uniforme, ou m ≤ 4 C'est exactement le terme dans L1 à L4
Bc — Huffman N · H(P̂) + D_huf P̂ non-uniforme et N > N* = D_huf / (log₂m − H) Remplace N·log₂m par N·H(P̂) < N·log₂m
Bd — Elias N · (log₂m + 2·log₂log₂m) m non connu au décodeur a priori Universel : proche entropie, sans coder m
Ab (Monochromatique) · C_color = 0 quel que soit B La couleur est l'indice du processus — aucun terme couleur, aucune marque explicite. L4_mono = log₂N + log₂C(r,N) uniquement.
Seuil N* — Huffman (Bc) vs Uniforme (Bb)
Bc est avantageux si :
N · H(P̂) + D_huf < N · log₂m

soit : N > N* = D_huf / (log₂m − H(P̂))

Exemple m=16, H=2.8 bits, D_huf=80 bits :
N* = 80 / (4 − 2.8) = 67 sauts

Au-delà de 67 sauts, Huffman compresse mieux que l'uniforme.
Condition Ba (séquentiel) < Bc (Huffman)
Ba est avantageux sur Bc si les transitions sont rares :

log₂m + N_trans·log₂m < N·H(P̂) + D_huf

soit : N_trans < (N·H(P̂) + D_huf − log₂m) / log₂m

Exemple N=80, m=16, H=2.8, D_huf=80 :
N_trans < (80·2.8 + 80 − 4) / 4 ≈ 75 transitions
→ Ba presque toujours avantageux sur processus couleur lents.
📐 Formules L1–L4 Généralisées selon le Mode B
Principe : L_i(B) = L_temporel_i + C_color(B). La partie temporelle est indépendante de B. Seul le terme couleur change. Le mode Bb (uniforme) redonne exactement les formules L1–L4 classiques.
Code L_temporel (indép. de B) + C_color(Ba) séq. + C_color(Bb) unif. ← L actuel + C_color(Bc) Huffman + C_color(Bd) Elias
L1 r log₂m + N_t·log₂m r · log₂m r · H(P̂) + D_huf r · L*(m)
L2 log₂N + N·log₂r log₂m + N_t·log₂m N · log₂(rm) N · H(P̂) + D_huf N · L*(m)
L3 r log₂m + N_t·log₂m N · log₂m → r + N·log₂m N · H(P̂) + D_huf N · L*(m)
L4 ★ log₂N + log₂C(r,N) log₂m + N_t·log₂m N · log₂m → L4 classique N · H(P̂) + D_huf N · L*(m)
L4_mono log₂N + log₂C(r,N) C_color = 0 — mode Ab monochromatique, couleur portée par l'indice du processus. Borne inférieure absolue, indépendante de B.
Formules compactes — mode Bc (Huffman)
L4(Bc) = log₂N + log₂C(r,N) + N·H(P̂) + D_huf
Gain(Bc vs Bb) = N·(log₂m − H(P̂)) − D_huf [> 0 si N > N*]
N* = D_huf / (log₂m − H(P̂)) [seuil de rentabilité Huffman]
Couleurs hiérarchiques avec Huffman (F × B = Bc)
C_color_hier(Bc, L niveaux) = N · Σᵢ₌₁ᴸ H(P̂ᵢ) + Σᵢ₌₁ᴸ D_huf_i
Si niveaux indépendants : H(m₁,…,mL) = ΣH(mᵢ) ≤ L·8 bits → C_color_hier(Bc) ≤ C_color_hier(Bb) = N·8L
Si niveaux corrélés : codage conditionnel → gain supplémentaire H(m₂|m₁) < H(m₂)

"""color_coding.py — Dimension B du cartouche ABCDEFGH Référence : J. Nembé, versatile_pp.tex, uniform_temp_coding.tex CORRECTION FONDAMENTALE : L_i(B) = L_temporel_i + C_color(B) Bb (uniforme) → C_color = N·log₂m ← terme des formules L1-L4 actuelles Ba (séquentiel) → C_color = log₂m + N_trans·log₂m Bc (Huffman) → C_color = N·H(P̂) + D_huf Bd (Elias) → C_color = N·L*(m_k) [m inconnu au décodeur] """ from math import log2, floor from typing import Literal import numpy as np ColorMode = Literal["Ba", "Bb", "Bc", "Bd"] # ── Utilitaires ────────────────────────────────────────────────────── def empirical_entropy(marks: np.ndarray, m: int) -> float: """H(P̂) — entropie empirique des marques couleur""" counts = np.bincount(marks, minlength=m).astype(float) probs = counts / counts.sum() return float(-np.sum(p * log2(p) for p in probs if p > 0)) def huffman_dict_overhead(m: int) -> int: """D_huf ≈ m·(⌊log₂m⌋+1) bits pour encoder le dictionnaire""" return m * (floor(log2(m)) + 1) def elias_delta_len(n: int) -> int: """Longueur du code δ d'Elias pour n ≥ 1""" k = floor(log2(n)) if n >= 1 else 0 return 1 + k + 2 * floor(log2(1 + k + 1e-9)) # ── Dimension B — 4 modes de codage couleur ────────────────────────── def color_cost(marks: np.ndarray, m: int, mode: ColorMode) -> float: """Coût de codage de la couleur selon le mode B. L_i(B) = L_temporel_i + color_cost(marks, m, mode_B) Ba — séquentiel : 1 terme en tête + transitions seulement Bb — uniforme : N·log₂m ← terme ACTUEL des formules L1–L4 Bc — Huffman : N·H(P̂) + D_huf (si N > N*, Bc < Bb) Bd — Elias : N·L*(m_k) si m non connu au décodeur """ N = len(marks) if mode == "Ba": # séquentiel N_trans = int(np.sum(marks[1:] != marks[:-1])) return log2(m) + N_trans * log2(m) elif mode == "Bb": # uniforme — L1–L4 actuels return N * log2(m) elif mode == "Bc": # Huffman H = empirical_entropy(marks, m) D = huffman_dict_overhead(m) return N * H + D elif mode == "Bd": # universel Elias return float(sum(elias_delta_len(int(c)+1) for c in marks)) def select_color_mode( marks: np.ndarray, m: int, m_known: bool = True ) -> tuple[ColorMode, dict]: """Retourne le mode B optimal et le tableau comparatif des coûts.""" modes = ["Ba", "Bb", "Bc"] + (["Bd"] if not m_known else []) costs = {b: color_cost(marks, m, b) for b in modes} best = min(costs, key=costs.get) return best, costs def huffman_threshold(m: int, H: float, D_huf: int = None) -> float: """N* au-delà duquel Huffman (Bc) bat l'uniforme (Bb).""" if D_huf is None: D_huf = huffman_dict_overhead(m) delta = log2(m) -
